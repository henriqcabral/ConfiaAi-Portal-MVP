Crie uma landing page funcional para o site 'Confia.ai', que se trata de uma ferramenta com foco na tradução de apólice de seguro de automóvel. A página deve ter um design moderno, com uma área de upload de arquivo (PDF) onde o usuário pode enviar sua apólice. A página deve usar inteligência artificial para ler o documento e exibir automaticamente as informações principais da apólice em linguagem clara e simples (como coberturas, franquia, exclusões e assistência 24h). Inclua CTAs claros para o usuário enviar o documento. O design deve ser intuitivo e responsivo para dispositivos móveis. faça o projeto em nojejs com react.

A pagina deve ser funcional implemente um chatbot utilizando as APIs da OpenAI e crie um arquivo .env para que possa colocar a API Key que será usada no chatbot.
